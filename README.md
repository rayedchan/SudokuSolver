SudokuSolver
============
9x9 Sudoku Board  
This program attempts to solve the puzzle using human-logic.  
  
Background  
Sudoku Coordinate System 9x9 Board  
  
     A B C  D E F   G H I     
   A      |       |    
   B  I   |  II   |  III    
   C      |       |    
    ------|-------|-------    
   D      |       |    
   E  IV  |   V   |  VI    
   F      |       |    
    ------|-------|-------    
   G      |       |    
   H  VII | VIII  |  IX    
   I      |       |    
  
   [Left - Right] = x coordinate  rows  
   [Up - Down] = y coordinate columns  
   The sudoku board is made of 81 coordinate points.
